# Security Policy

This repository hosts a static personal CV deployed to Cloudflare Workers. There
is no backend and no user data — the site is fully static.

## Reporting a vulnerability

If you find a security issue with the site or this repository, please report it
privately by email to **gustaw.beznicki@gmail.com**. Please do not open a public
issue for security-sensitive reports.

I'll aim to acknowledge reports within a few days.

## Scope

- The deployed site: https://gustawbeznicki.dev
- This repository's build and deploy configuration (GitHub Actions, Wrangler)
